# PastDrift 时光漂移

> 一个现代人穿越到古代架空朝代的故事

## 故事简介

现代都市青年意外穿越，落入一个架空的古代王朝。

这里没有历史课本里的朝代，却有着似曾相识的风土人情。

他带着未来的知识与思维，在这个陌生的世界一步步站稳脚跟——却也一步步陷入更深的漩涡。
