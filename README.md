# PastDrift / 时光漂移

> 一个现代人穿越到古代架空朝代的故事

---

## 📖 故事简介

**2024年，陆鹤鸣，27岁。**

国内顶尖量化私募的投资经理，数学竞赛保送TOP2，常春藤金融工程硕士。手下管理的百亿量化基金年化收益跑赢大盘数个点。在同行眼中，他是那个能用数学模型精确预测市场波动的“天才”。

只有他自己知道——他的心脏早已不堪重负。

又一个彻夜盯盘跑因子的凌晨，连续工作数日的身体达到极限。心脏传来一阵剧烈的绞痛——**在量化模型跑出史上最优夏普比率的那一刻，他的心跳归零了。**

再醒来时，他躺在一辆颠簸的马车里，穿着粗布麻衣，头痛欲裂。身边只有一个约莫七八岁的小丫头，怯生生地叫他：

**“公子，您终于醒了。”**

这里是“大梁王朝”——一个从未出现在任何史书上的架空国度。

都城长安里，世家门阀把持着盐铁生意，粮价年年被操纵，钱庄的利息比高利贷还狠，市面上的银铜兑换率每天都在变。这套“市场”，在陆鹤鸣眼中——**简直处处是套利空间。**

他带着未来的知识、量化的思维，在这个陌生的世界一步步站稳脚跟——却也一步步陷入更深的漩涡。

当大梁朝堂，只是棋盘。当资本之手，搅动天下。
